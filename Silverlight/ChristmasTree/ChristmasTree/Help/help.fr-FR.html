ChristmasTree 1.1.0

Développeur: Natalia Derevenets <nna911@inbox.ru>

ChristmasTree est une application divertissante pour les dispositifs sur la base de Windows Phone qui vous permet de décorer l'arbre du Nouvel an directement dans votre téléphone.

Si vous avez des questions de l’utilisation de cette application, vous pouvez contacter le développeur par cette Website: http://christmastree.sourceforge.net/.

Comment décorer l'arbre du Nouvel an

1. Lancez l’application et appuyez sur le bouton pour choisir l’image de fond et la vue de votre sapin, et pour régler le volume de la musique.

2. Appuyez sur le bouton pour ouvrir la galerie des jouets de Nouvel an.

3. Choisissez un jouet de la liste par par une pression longue du doigt et placez le sur l’arbre. Placez le jouet derrière les branches ou à l’avant-plan par une pression longue du doigt. Déplacez le jouet au-dela de l’arbre de Noël pour le supprimer.

4. Répétez le pas précédent pour choisir les autres jouets de la galerie. Ayant décoré l’arbre de Nouvel an complètement, appuyez sur le bouton pour sauvgarder son image dans la galerie du téléphone.

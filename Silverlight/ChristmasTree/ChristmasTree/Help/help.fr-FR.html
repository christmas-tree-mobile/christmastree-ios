ChristmasTree 1.3.0

Développeur: Natalia Derevenets <nna911@inbox.ru>

ChristmasTree est une application de divertissements pour les appareils basés sur Windows Phone, qui vous permet de décorer un arbre de Noël directement sur votre téléphone.

Si vous avez des questions sur l'utilisation de cette application, vous pouvez contacter le développeur via ce site Web: http://christmastree.sourceforge.net/.

Comment décorer un arbre de Noël

1. Lancez l'application et appuyez sur pour sélectionner une image d'arrière-plan et l'apparence de votre arbre de Noël, ainsi que pour régler le volume de la musique.

2. Appuyez sur pour ouvrir la galerie de décorations de Noël.

3. Sélectionnez votre décoration préférée dans la galerie en l'appuyant longuement avec un doigt, puis déplacez-la sur l'arbre. Déplacez la décoration derrière les branches ou à l'avant en l'appuyant longuement avec un doigt. Déplacez la décoration en dehors de l'arbre pour la supprimer de l'écran.

4. Répétez l'étape précédente, en choisissant des décorations différentes dans la galerie. Une fois votre arbre est entièrement décoré, appuyez sur pour enregistrer son image dans la galerie du téléphone.
